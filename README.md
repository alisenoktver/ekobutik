# АлисЁноК — одностраничный сайт бренда

Готовый адаптивный лендинг для бренда «АлисЁноК» на React + Vite + Tailwind CSS.

## Что внутри

- Hero-блок с фирменным позиционированием и CTA
- Блок «О бренде» с иконками и плавным появлением
- Каталог из 4 ключевых категорий товаров
- Галерея в Pinterest-style раскладке
- Преимущества и отзывы
- Footer с ссылками на маркетплейсы и соцсети
- SEO-мета, alt-тексты, favicon, семантическая верстка

## Структура

- `index.html` — базовый HTML с SEO и подключением шрифтов
- `src/App.tsx` — структура и контент страницы
- `src/index.css` — стили, анимации и палитра
- `public/images` — изображения и favicon

## Запуск локально

1. Установите зависимости:
   `npm install`
2. Запустите dev-сервер:
   `npm run dev`
3. Откройте адрес из терминала.

## Сборка

1. Соберите проект:
   `npm run build`
2. Готовые статические файлы будут в папке `dist`.

## Публикация на GitHub Pages

1. Загрузите репозиторий на GitHub.
2. Настройте деплой папки `dist` через GitHub Actions или другой статический деплой.
3. Сайт не требует backend и базы данных.
